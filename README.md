# smart-scheduling-assistant

基于大模型的智能生产排产助手

项目经历：我曾参与“智能排产助手”项目，负责为一家中小型制造企业设计基于大模型的SaaS产品。通过深入车间调研，我们梳理出订单延误、设备闲置等核心痛点，并利用GPT-4构建了一个能自动解析生产约束、生成排产方案的智能体。我主导了历史工单数据的标注与处理，优化了模型的约束参数，使排产方案接受率从初期60%提升至85%。产品上线后，客户平均订单交付周期缩短了18%，该项目让我对B端AI产品的场景落地有了扎实理解

## 运行
```bash
pip install -r requirements.txt
python main.py
```
